# ShopGrid — E-commerce Product Listing

A responsive product listing web app built with HTML, CSS, and JavaScript.

## Features

- Fetches real product data from [FakeStore API](https://fakestoreapi.com)
- Search products by name or description
- Filter by category
- Sort by price (low to high / high to low)
- Add to cart with live counter
- Fully responsive — works on mobile and desktop

## Tech Stack

- HTML5
- CSS3 (CSS Variables, Flexbox, Grid)
- Vanilla JavaScript (Fetch API, DOM manipulation)
- [FakeStore REST API](https://fakestoreapi.com)

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/ecommerce-product-listing.git
   ```
2. Open `index.html` in your browser — no build step needed.
